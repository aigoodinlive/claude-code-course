# שיעור 6: מה זה Skills? (יכולות מיוחדות)

**משך:** 20 דקות
**רמה:** מתחילים

---

## מה זה Skills?

### ההסבר הפשוט

**Skills הם "כוחות על" שאפשר להוסיף ל-Claude Code.**

כמו אפליקציות בטלפון - מתקינים רק את מה שצריכים.

### אנלוגיה: ארגז כלים

דמיינו ארגז כלים:

| Claude Code בסיסי | Claude Code עם Skills |
|-------------------|----------------------|
| פטיש ומברג | פטיש, מברג, מקדחה, מסור, מפלס... |
| עושה דברים בסיסיים | עושה דברים מתקדמים |

**Skills = כלים נוספים לארגז**

---

## דוגמאות ל-Skills

### Skill: יצירת קומיט
**מה עושה:** שומר שינויים ב-Git (גרסאות)
**איך משתמשים:** `/commit`

### Skill: סריקת אתרים
**מה עושה:** קורא מידע מאתרי אינטרנט
**איך משתמשים:** `/web-fetch`

### Skill: יצירת PDF
**מה עושה:** ממיר קבצים ל-PDF
**איך משתמשים:** `/pdf`

### Skill: שליחת הודעות
**מה עושה:** שולח הודעות וואטסאפ
**איך משתמשים:** `/whatsapp`

---

## איך יודעים אילו Skills יש?

### הפקודה הקסומה:

```
/skills
```

Claude יציג רשימה של כל ה-Skills הזמינים.

### דוגמה לפלט:

```
Available Skills:
- /commit - שמירת שינויים ב-Git
- /help - עזרה
- /review-pr - סקירת קוד
```

## איך משתמשים ב-Skill?

### פשוט מאוד!

הקלידו `/` ואז את שם ה-Skill:

```
/commit
```

או

```
/pdf
```

### Claude ינחה אתכם

אחרי שמקלידים את הפקודה, Claude ישאל שאלות אם צריך:

```
> /pdf

Claude: איזה קובץ אתה רוצה להמיר ל-PDF?
```

## מבנה של Skill

### איפה Skills נמצאים?

```
הפרויקט-שלי/
├── CLAUDE.md
├── .claude/
│   └── skills/
│       └── my-skill/
│           └── SKILL.md    ← הקובץ של ה-Skill
└── ...
```

### מה יש בקובץ SKILL.md?

```markdown
# שם ה-Skill

## תיאור
מה ה-Skill עושה

## איך להשתמש
/skill-name

## פרמטרים
- param1: מה זה
- param2: מה זה

## דוגמה
/skill-name param1 param2
```

## למה Skills שימושיים?

### 1. חוסכים זמן

במקום להסביר כל פעם מה לעשות:
```
> /commit
```

עובד מיד!

### 2. עקביות

ה-Skill תמיד עושה את אותו הדבר באותה צורה.

### 3. שיתוף

אפשר לשתף Skills עם אחרים:
- "הנה ה-Skill שאני משתמש בו לדוחות"
