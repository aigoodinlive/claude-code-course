## חלק א': איך לשאול שאלות נכון

### הכלל הזהב

**ככל שהשאלה יותר ברורה - התשובה יותר טובה.**

### שאלות לא טובות ❌

```
"תעזור לי"
"משהו לא עובד"
"תעשה את זה יותר טוב"
```

Claude לא יודע מה אתם רוצים!

### שאלות טובות ✅

```
"תראה לי את כל קבצי ה-PDF בתיקייה הזו"
"יש לי קובץ Excel עם לקוחות, תייצר ממנו רשימת מיילים"
"תסביר לי מה עושה הקובץ הזה"
```

Claude מבין בדיוק מה לעשות!

### הנוסחה לשאלה מושלמת

```
[מה יש לי] + [מה אני רוצה] + [פרטים נוספים]
```

**דוגמאות:**

| מה יש לי | מה אני רוצה | פרטים |
|----------|-------------|-------|
| 50 תמונות | לארגן לתיקיות | לפי תאריך |
| קובץ טקסט עם משימות | להפוך לטבלה | עם עמודות: משימה, סטטוס, תאריך |
| תיקייה מבולגנת | לנקות | למחוק קבצים ישנים מלפני 2020 |

## חלק ב': פקודות חשובות

### פקודות מתחילות ב-/

| פקודה | מה עושה | מתי להשתמש |
|--------|---------|------------|
| `/help` | מציג עזרה | כשלא יודעים מה לעשות |
| `/exit` | יוצא מ-Claude Code | סיום עבודה |
| `/clear` | מנקה את השיחה | התחלה מחדש |
| `/resume` | חוזר לשיחה קודמת | המשך עבודה |

### הדגמה

```
> /help

Claude מציג:
- רשימת פקודות
- קיצורי מקלדת
- טיפים שימושיים
```

```
> /exit

Claude: "להתראות! השיחה נשמרה."
```

### לראות מה יש

```
"תראה לי מה יש בתיקייה"
"מצא את כל קבצי ה-Excel"
"כמה קבצים יש פה?"

### לקרוא קובץ

```

"מה כתוב בקובץ ההגדרות?"

### ליצור ולערוך

```

"הוסף שורה לסוף הקובץ"
"שנה את המילה 'ישן' ל'חדש' בכל הקובץ"
```

## חלק ד': יציאה וחזרה

### למה זה חשוב?

- אפשר להמשיך עבודה ממחר
- Claude זוכר את ההקשר
- חוסך זמן

### איך יוצאים?

```
/exit
```

או פשוט:
```
exit
```

או סוגרים את הטרמינל.

### איך חוזרים?

1. פתחו טרמינל
2. נווטו לתיקייה שעבדתם בה
3. הקלידו `claude`
4. הקלידו `/resume`

Claude יראה רשימה של שיחות אחרונות - בחרו לאיזו לחזור.

## חלק ה': דברים חשובים לזכור

### 1. Claude שואל לפני פעולות רגישות

```
Claude: "אני עומד למחוק 5 קבצים. להמשיך? (y/n)"
```

**y** = כן, תמשיך
**n** = לא, תעצור

**תמיד קראו לפני שמאשרים!**

### 2. אפשר לעצור בכל רגע

```
Ctrl + C
```

זה עוצר כל פעולה. תמיד עובד.

### 3. Claude לא יודע הכל

אם הוא לא בטוח - הוא יגיד.
אל תניחו שהוא תמיד צודק.

### 4. אפשר לתקן

```
> לא, התכוונתי למשהו אחר. אני רוצה ש...
```

Claude ישמח לנסות שוב.

## ✅ תרגיל מעשי

### משימה 1: ראו מה יש (3 דקות)

פתחו Claude Code בתיקיית Documents ושאלו:
```
מה יש בתיקייה הזו?
```

### משימה 2: צרו קובץ (3 דקות)

```
צור קובץ בשם test.txt עם הטקסט "שלום עולם"
```

אחר כך:
```
תראה לי מה כתוב בקובץ
```

### משימה 3: יציאה וחזרה (3 דקות)

1. הקלידו `/exit`
2. פתחו Claude Code שוב
3. הקלידו `/resume`
4. וודאו שהשיחה הקודמת חזרה
