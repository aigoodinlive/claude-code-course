## חלק א': התקנת Node.js

### שלב 1: פתיחת האתר

1. פתחו את הדפדפן (Chrome, Edge, Firefox - לא משנה)
2. הקלידו בשורת הכתובת: **nodejs.org**
3. לחצו Enter

### שלב 2: הורדת הקובץ

**מה תראו באתר:**
- שני כפתורים ירוקים גדולים
- בחרו את הכפתור השמאלי שכתוב עליו **"LTS"**

**מה זה LTS?**
Long Term Support = גרסה יציבה ומומלצת

**לחצו על הכפתור** - הקובץ יתחיל להוריד.

### שלב 3: הרצת ההתקנה

1. פתחו את התיקייה **Downloads** (הורדות)
2. חפשו קובץ בשם כמו: `node-v20.XX.X-x64.msi`
3. **לחצו עליו פעמיים** להפעלה

### שלב 4: אשף ההתקנה

יפתח חלון התקנה. עקבו אחרי השלבים:

**מסך 1: "Welcome to Node.js Setup"**
→ לחצו **Next**

**מסך 2: "License Agreement"**
→ סמנו "I accept..." ולחצו **Next**

**מסך 3: "Destination Folder"**
→ השאירו את ברירת המחדל, לחצו **Next**

**מסך 4: "Custom Setup"**
→ השאירו הכל כמו שזה, לחצו **Next**

**מסך 5: "Tools for Native Modules"**
→ **אל תסמנו כלום** (לא צריך), לחצו **Next**

**מסך 6: "Ready to install"**
→ לחצו **Install**

**ייתכן שיבקשו הרשאת מנהל** - לחצו "כן" (Yes)

**מסך 7: "Completed"**
→ לחצו **Finish**

### שלב 5: בדיקה שההתקנה הצליחה

1. פתחו **טרמינל חדש** (חשוב - חדש!)
   - Windows: חפשו "PowerShell"
   - Mac: חפשו "Terminal"

2. הקלידו:
```
node --version
```

3. לחצו Enter

**מה צריך לראות:**
```
v20.10.0
```
(המספרים יכולים להיות קצת שונים)

**אם ראיתם מספר גרסה - מצוין! Node.js מותקן!**

**אם ראיתם שגיאה:**
- סגרו את הטרמינל
- פתחו טרמינל חדש
- נסו שוב

## חלק ב': התקנת Claude Code

### שלב 1: פתיחת טרמינל

פתחו PowerShell (Windows) או Terminal (Mac)

### שלב 2: הפקודה הקסומה

הקלידו את הפקודה הבאה **בדיוק כמו שהיא:**

```
npm install -g @anthropic-ai/claude-code
```

**מה זה אומר?**
- `npm` = מנהל החבילות של Node.js
- `install` = התקן
- `-g` = גלובלי (שיעבוד מכל מקום)
- `@anthropic-ai/claude-code` = שם התוכנה

### שלב 3: המתנה

תראו הרבה טקסט רץ על המסך - **זה נורמלי!**

ההתקנה יכולה לקחת 1-3 דקות.

**מה תראו:**
```
added 150 packages in 45s
```
(המספרים יכולים להשתנות)

### שלב 4: בדיקה

הקלידו:
```
claude --version
```

**מה צריך לראות:**
```
Claude Code v1.x.x
```

**הצלחתם! Claude Code מותקן!**

## 🔧 פתרון בעיות נפוצות

### "npm לא מזוהה כפקודה"

**הבעיה:** Node.js לא מותקן נכון

**פתרון:**
1. סגרו את הטרמינל
2. התקינו Node.js מחדש
3. **הפעילו מחדש את המחשב**
4. פתחו טרמינל חדש ונסו שוב

### "אין לי הרשאות" / "Permission denied"

**Windows:**
1. חפשו "PowerShell"
2. קליק ימני → "Run as Administrator"
3. נסו את ההתקנה שוב

**Mac:**
הוסיפו `sudo` לפני הפקודה:
```
sudo npm install -g @anthropic-ai/claude-code
```
(יבקש סיסמה - הקלידו אותה)

### "הדפדפן לא נפתח להתחברות"

**פתרון:**
1. העתיקו את הקישור שמופיע בטרמינל
2. הדביקו אותו בדפדפן ידנית

### "אני לא רואה שום דבר אחרי ההתקנה"

**זה נורמלי!** אם אין הודעת שגיאה - ההתקנה הצליחה.
נסו להריץ `claude --version` לבדיקה.
