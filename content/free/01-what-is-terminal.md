## מה זה בכלל טרמינל?

### ההסבר הפשוט

הטרמינל הוא **דרך לדבר עם המחשב בכתיבה**.

במקום ללחוץ על כפתורים ואייקונים, אתם **כותבים פקודות** והמחשב מבצע אותן.

### אנלוגיה: מסעדה

דמיינו מסעדה:

| דרך רגילה (GUI) | טרמינל |
|-----------------|--------|
| מסתכלים על התפריט עם תמונות | אומרים למלצר בדיוק מה רוצים |
| לוחצים על כפתור "הזמן" | כותבים את ההזמנה |
| ממתינים שהמערכת תעבד | מקבלים תשובה מיידית |

שני האופנים עובדים - הטרמינל פשוט יותר ישיר.

## איך הטרמינל נראה?

### ב-Windows: זה נקרא PowerShell או Command Prompt

```
PS C:\Users\יוסי>
```

מה אנחנו רואים כאן?
- `PS` = PowerShell (סוג הטרמינל)
- `C:\Users\יוסי` = התיקייה שאנחנו נמצאים בה
- `>` = כאן כותבים את הפקודה

### ב-Mac/Linux: זה נקרא Terminal

```
yossi@macbook ~ %
```

- `yossi` = שם המשתמש
- `macbook` = שם המחשב
- `~` = התיקייה הראשית שלכם
- `%` = כאן כותבים את הפקודה

## 🖥️ איך פותחים את הטרמינל?

### Windows (הכי קל!)

**אפשרות 1: חיפוש**
1. לחצו על מקש Windows (הכפתור עם החלון)
2. הקלידו: `powershell`
3. לחצו Enter

**אפשרות 2: קיצור מקלדת**
1. לחצו `Windows + R` (יפתח חלון "הפעלה")
2. הקלידו: `powershell`
3. לחצו Enter

**אפשרות 3: קליק ימני**
1. לחצו קליק ימני על כפתור Start
2. בחרו "Windows PowerShell"

### Mac

**אפשרות 1: Spotlight**
1. לחצו `Cmd + Space`
2. הקלידו: `terminal`
3. לחצו Enter

**אפשרות 2: דרך התיקיות**
1. פתחו Finder
2. לכו ל: Applications → Utilities → Terminal

## 🎮 ננסה ביחד! (תרגול מעשי)

### פתחו את הטרמינל עכשיו

עצרו וּפתחו את הטרמינל לפי ההוראות למעלה.

**מוכנים? מצוין!**

### פקודה ראשונה: "איפה אני?"

הקלידו את הפקודה הזו ולחצו Enter:

**Windows:**
```
pwd
```

**Mac/Linux:**
```
pwd
```

**מה קיבלתם?**
משהו כמו: `C:\Users\YourName` או `/Users/YourName`

זו התיקייה שאתם נמצאים בה עכשיו!

`pwd` = "Print Working Directory" = "הדפס את התיקייה הנוכחית"

### פקודה שנייה: "מה יש פה?"

הקלידו:

**Windows:**
```
dir
```

**Mac/Linux:**
```
ls
```

**מה קיבלתם?**
רשימה של כל הקבצים והתיקיות במקום שאתם נמצאים בו!

### פקודה שלישית: "תגיד שלום"

הקלידו:
```
echo Hello World
```

**מה קיבלתם?**
```
Hello World
```

המחשב פשוט חזר על מה שאמרתם לו!

---

## 🔑 5 פקודות בסיסיות שכדאי לזכור

| פקודה | מה עושה | דוגמה |
|-------|---------|-------|
| `pwd` | מראה איפה אני | `pwd` |
| `ls` / `dir` | מראה מה יש בתיקייה | `ls` |
| `cd` | עובר לתיקייה אחרת | `cd Documents` |
| `echo` | מדפיס טקסט | `echo שלום` |
| `clear` / `cls` | מנקה את המסך | `clear` |

**טיפ:** `cd ..` (עם שתי נקודות) = חזרה לתיקייה הקודמת

## 😰 מה לעשות אם משהו השתבש?

### "הפקודה לא עובדת"

**בדקו:**
- שגיאות כתיב (המחשב מדייק!)
- רווחים מיותרים
- אות גדולה/קטנה (ב-Mac/Linux זה משנה)

### "נתקעתי ולא יודע מה לעשות"

**פתרון:** לחצו `Ctrl + C`

זה עוצר כל פעולה שרצה. תמיד עובד.

### "רוצה להתחיל מחדש"

**Windows:** הקלידו `cls` ולחצו Enter
**Mac/Linux:** הקלידו `clear` ולחצו Enter
