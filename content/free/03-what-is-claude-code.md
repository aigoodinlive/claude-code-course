# שיעור 3: מה זה Claude Code?

**משך:** 20 דקות
**רמה:** מתחילים

---

## התשובה בשני משפטים

**Claude Code הוא עוזר AI שגר במחשב שלכם.**

הוא לא רק עונה על שאלות - הוא **עושה דברים**: קורא קבצים, כותב קוד, מריץ פקודות, ומארגן את העבודה שלכם.

---

## מה ההבדל מ-ChatGPT?

### ChatGPT = שיחה בדפדפן

אתם פותחים אתר, כותבים שאלה, מקבלים תשובה.
אם רוצים שהוא יכתוב משהו - צריך **להעתיק ולהדביק**.

### Claude Code = עוזר שיושב לידכם

הוא רואה את הקבצים שלכם, יכול לערוך אותם, ולהריץ פקודות.
אין צורך להעתיק-הדביק - הוא **עושה את העבודה ישירות**.

### טבלת השוואה

| ChatGPT | Claude Code |
|---------|-------------|
| עובד בדפדפן | עובד בטרמינל |
| רואה רק מה שמדביקים לו | רואה את כל הקבצים שלכם |
| נותן תשובות | מבצע פעולות |
| צריך להעתיק/להדביק | עורך ישירות |
| שיחה אחת | זוכר פרויקטים |

---

## דוגמה מהחיים

### הבעיה: תיקייה מבולגנת

יש לכם תיקיית Downloads עם 200 קבצים מעורבבים.

### עם ChatGPT:
```
אתם: "תכתוב לי סקריפט שמארגן קבצים לפי סוג"
ChatGPT: "הנה הקוד: [קוד ארוך]"
אתם: מעתיקים... פותחים קובץ... מדביקים... שומרים... מריצים...
```

**5 דקות של העתק-הדבק**

### עם Claude Code:
```
אתם: "תארגן לי את הקבצים בתיקייה הזו לפי סוג"
Claude Code: *מסתכל* *יוצר תיקיות* *מעביר קבצים*
"סיימתי! העברתי 47 תמונות, 23 מסמכים, ו-15 קבצי וידאו"
```

**30 שניות**

---

## מה Claude Code יכול לעשות?

### 📂 עבודה עם קבצים
- לקרוא מסמכים
- לחפש מידע בתוך קבצים
- לארגן ולמיין
- לשנות שמות

### 📝 יצירת תוכן
- לכתוב מיילים
- לסכם פגישות
- ליצור דוחות
- לתרגם

### 💻 אוטומציה
- להריץ משימות חוזרות
- לעבד נתונים
- ליצור גיבויים

### 🔍 ניתוח
- למצוא דפוסים במידע
- לסכם מסמכים ארוכים
- להשוות בין קבצים

---

## מה Claude Code לא יכול לעשות?

בואו נהיה כנים:

- ❌ **לא יכול לגלוש באינטרנט** (מוגבל למחשב שלכם)
- ❌ **לא יכול לשלוח מיילים** (צריך אינטגרציה)
- ❌ **לא תמיד צודק** (תמיד בדקו!)
- ❌ **לא יודע מידע מעודכן** (תאריך הידע שלו מוגבל)
