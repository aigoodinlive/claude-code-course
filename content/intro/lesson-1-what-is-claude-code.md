## 📖 מה זה Claude Code?

### ההגדרה הפשוטה
Claude Code הוא **עוזר AI שחי בתוך המחשב שלכם**.

בניגוד ל-ChatGPT שאתם מדברים איתו בדפדפן, Claude Code:
- רואה את הקבצים שלכם
- מריץ פקודות
- כותב ועורך קוד
- עובד **איתכם** - לא רק עונה לשאלות

### האנלוגיה
דמיינו שיש לכם עובד חדש שיושב לידכם:
- הוא יכול לראות את המסך שלכם
- הוא יודע לעבוד עם כל התוכנות
- הוא לא נעלב ולא מתעייף
- הוא זוכר הכל מהשיחות הקודמות

זה Claude Code.

---

## 🔄 ההבדל מ-ChatGPT

| ChatGPT | Claude Code |
|---------|-------------|
| עובד בדפדפן | עובד בטרמינל/מחשב |
| רואה רק מה שמדביקים לו | רואה את כל הקבצים |
| נותן תשובות | מבצע פעולות |
| צריך להעתיק/להדביק | עורך ישירות |
| שיחה אחת | זוכר פרויקטים |

### דוגמה מעשית

**ChatGPT:**
```
אתה: "תכתוב לי סקריפט שמארגן קבצים"
ChatGPT: "הנה הקוד: ... עכשיו תעתיק אותו לקובץ..."
```

**Claude Code:**
```
אתה: "תארגן לי את הקבצים בתיקייה הזו לפי סוג"
Claude Code: *מסתכל על הקבצים*
              *יוצר תיקיות*
              *מעביר את הקבצים*
              "סיימתי! יצרתי 5 תיקיות והעברתי 47 קבצים"
```

## 💼 למה עסקים קטנים צריכים את זה?

### 1. חיסכון בזמן
- משימות של שעות → דקות
- אוטומציה של דברים חוזרים
- פחות העתק-הדבק

### 2. לא צריך מתכנת
- Claude Code כותב את הקוד
- אתם רק מסבירים מה אתם רוצים
- הוא מתקן את עצמו אם יש בעיות

### 3. עקביות
- תמיד עושה את אותו הדבר נכון
- לא שוכח שלבים
- מתעד הכל

### דוגמאות מהעולם האמיתי
- **רואה חשבון:** "תייצר דוח מהקבצים האלה"
- **סוכן נדל"ן:** "תעדכן את כל הנכסים באתר"
- **מעצב:** "תארגן את כל התמונות לפי לקוח"
- **יועץ:** "תסכם את כל הפגישות מהשבוע"

## 🛠️ התקנה

### דרישות מקדימות
- מחשב עם Windows/Mac/Linux
- חיבור לאינטרנט
- Node.js (נתקין ביחד)

### שלב 1: התקנת Node.js
```bash
# Windows - הורידו מ:
# https://nodejs.org/
# בחרו LTS (גרסה יציבה)

# Mac (עם Homebrew):
brew install node
```

### שלב 2: התקנת Claude Code
```bash
npm install -g @anthropic-ai/claude-code
```

### שלב 3: הפעלה ראשונה
```bash
claude
```

יופיע מסך כניסה - התחברו עם חשבון Anthropic.
