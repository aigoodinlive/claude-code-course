## 🌅 שגרת בוקר עם Claude

### הפקודה הראשונה ביום
```
מה המשימות שלי היום?
תסתכל על:
- הפגישות ביומן
- המיילים החשובים
- המשימות שנשארו מאתמול
```

### לפני כל פגישה
```
הכן לי תקציר על [שם לקוח/פרויקט]:
- מה עשינו עד עכשיו
- נקודות פתוחות
- מה צריך להחליט היום
```

### בסוף היום
```
סכם את מה שעשיתי היום וצור רשימת משימות למחר
```

---

## 📁 ניהול קבצים (5 דקות לכל משימה)

### תיקיית Downloads מבולגנת
```
ארגן את תיקיית Downloads:
- תמונות → Images/
- מסמכים → Documents/
- וידאו → Videos/
- שאר → Other/
מחק קבצים כפולים
```

### ארגון חשבוניות
```
מיין את כל החשבוניות בתיקייה:
- שנה שמות ל: חשבונית_[לקוח]_[תאריך]_₪[סכום].pdf
- צור תיקיות לפי חודש
- צור קובץ סיכום Excel
```

### גיבוי פרויקט
```
צור גיבוי של הפרויקט הזה:
- העתק לתיקיית backup עם תאריך
- דחוס ל-ZIP
- מחק גיבויים ישנים מעל 30 יום
```

## 📧 ניהול אימיילים

### ניקוי תיבת דואר
```
עבור על תיבת הדואר שלי:
- ניוזלטרים ושיווק → מחק
- הודעות מעל שנה → ארכיון
- לקוחות → סמן חשוב
- כסף/חוזים → תייק

צור סיכום: כמה נמחק, כמה נשאר
```

### תשובה מקצועית
```
קיבלתי את המייל הזה: [הדבק מייל]

כתוב תשובה מקצועית ש:
- מודה על הפנייה
- עונה על השאלה
- מציע צעד הבא
- קצר וענייני
```

### תבנית הצעת מחיר
```
צור תבנית הצעת מחיר ב-HTML:
- לוגו למעלה
- פרטי לקוח
- טבלת שירותים ומחירים
- תנאי תשלום
- חתימה
```

## 📊 ניתוח ומחקר

### מחקר מתחרים
```
נתח את המתחרים האלה: [רשימה]

לכל אחד מצא:
1. טווח מחירים
2. שירותים עיקריים
3. קהל יעד
4. יתרונות וחסרונות

צור טבלת השוואה
```

### ניתוח לקוחות
```
יש לי קובץ עם נתוני לקוחות.
נתח ומצא:
- מאיפה מגיעים הכי הרבה לקוחות?
- מה הגודל הממוצע של עסקה?
- מי הלקוחות הכי רווחיים?
- מגמות לאורך זמן
```

### סיכום פגישות
```
הנה הערות מהפגישה: [הדבק]

צור סיכום מסודר:
- נקודות עיקריות (3-5)
- החלטות שהתקבלו
- משימות + אחראי + דדליין
- נושאים לפגישה הבאה
```

## 💼 לפי מקצוע

### למעצבים
```
ארגן את תיקיית הפרויקטים:
- קבץ לפי לקוח
- בתוך כל לקוח: לפי פרויקט
- מצא וסמן גרסאות סופיות
- זהה קבצים כפולים
```

### לכותבים/יוצרי תוכן
```
נתח את הפוסטים שלי מהחודש האחרון:
- מה קיבל הכי הרבה תגובות?
- אילו נושאים מעניינים את הקהל?
- מה האורך האופטימלי?
- הצע 5 רעיונות לפוסטים הבאים
```

### ליועצים
```
צור תבנית דוח ייעוץ:
- סיכום מנהלים (חצי עמוד)
- מצב קיים
- ממצאים עיקריים
- המלצות (מתועדפות)
- צעדים הבאים
- נספחים
```

### לרואי חשבון / מנהלי חשבונות
```
עבור על קבצי ההנהלת חשבונות:
- מיין הוצאות לפי קטגוריה
- זהה חריגות מהממוצע
- סמן פריטים לבדיקה
- צור דוח חודשי מסכם
```

## 🔄 אוטומציה יומית

### משימות שחוזרות = אוטומציה
אם אתה עושה משהו 3 פעמים בשבוע:
```
צור סקריפט שעושה את זה אוטומטית:
[תאר את המשימה]
```

### דוגמאות לאוטומציה
```bash
# גיבוי יומי בשעה 18:00
"גבה את כל הפרויקטים הפעילים ל-backup/"

# דוח שבועי בכל יום ראשון
"צור סיכום שבועי של שעות עבודה"

# ניקוי חודשי
"מחק קבצים זמניים ונקה מקום"
```

## 🎓 טיפים למתחילים

### 1. התחל קטן
אל תנסה להפוך הכל לאוטומטי ביום אחד.
התחל ממשימה אחת קטנה ביום.

### 2. שמור את מה שעובד
כשמשהו עובד טוב - שמור את הפרומפט!
```
צור קובץ prompts.md עם הפרומפטים המוצלחים שלי
```

### 3. למד מטעויות
אם Claude עשה משהו לא נכון:
```
זה לא מה שרציתי. רציתי [הסבר].
בפעם הבאה שאני מבקש X, תעשה Y.
```

### 4. בנה הרגלים
- **בוקר:** סקירת משימות
- **לפני פגישה:** הכנת תקציר
- **אחרי פגישה:** סיכום ומשימות
- **סוף יום:** תכנון מחר
