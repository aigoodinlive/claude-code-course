**משך:** 30 דקות
**מטרה:** להכיר את הפקודות החשובות ולהרגיש בנוח עם Claude Code

---

## 🎯 מטרות השיעור

בסוף השיעור תדעו:
- איך לפתוח פרויקט
- את הפקודות הכי חשובות
- איך לשאול שאלות נכון
- איך לצאת ולחזור לשיחה

---

## 🚀 פתיחת פרויקט

### הדרך הפשוטה
```bash
# נווטו לתיקייה שלכם
cd C:\Users\שם\Documents\הפרויקט-שלי

# הפעילו את Claude Code
claude
```

### מה קורה כש-Claude נפתח?
1. הוא סורק את התיקייה
2. קורא קבצי הגדרות (אם יש)
3. מבין את ההקשר של הפרויקט
4. מוכן לעזור!

---

## 📋 פקודות חשובות (Slash Commands)

### פקודות ניהול

| פקודה | מה עושה | מתי להשתמש |
|--------|---------|------------|
| `/help` | מציג עזרה | כשלא יודעים מה לעשות |
| `/exit` | יוצא מ-Claude Code | סיום עבודה |
| `/clear` | מנקה את השיחה | התחלה מחדש |
| `/resume` | חוזר לשיחה קודמת | המשך עבודה |

### פקודות מידע

| פקודה | מה עושה |
|--------|---------|
| `/skills` | מציג skills טעונים |
| `/status` | סטטוס הפרויקט |
| `/config` | הגדרות |

### דוגמה מעשית
```
> /help
Claude מציג את כל הפקודות הזמינות

> /skills
Claude מראה אילו יכולות מיוחדות זמינות
```

---

## 💬 איך לשאול שאלות נכון

### ❌ שאלות לא טובות
```
"תעזור לי"
"משהו לא עובד"
"תעשה את זה יותר טוב"
```

### ✅ שאלות טובות
```
"תראה לי את כל קבצי ה-PDF בתיקייה הזו"
"יש לי קובץ Excel עם לקוחות, תייצר ממנו רשימת מיילים"
"תסביר לי מה עושה הקובץ config.js"
```

### הנוסחה לשאלה טובה
```
[מה יש לי] + [מה אני רוצה] + [איפה/מתי]
```

**דוגמאות:**
- "יש לי 50 תמונות, תארגן אותן לתיקיות לפי תאריך"
- "יש לי קובץ טקסט עם משימות, תהפוך אותו לטבלה"
- "הקוד הזה לא עובד, תמצא את הבעיה"

---

## 🔄 לצאת ולחזור

### יציאה מסודרת
```bash
/exit
# או פשוט
exit
```

### חזרה לשיחה קודמת
```bash
claude
# ואז:
/resume
```

Claude יראה לכם רשימה של שיחות אחרונות - בחרו לאיזו לחזור.

### למה זה חשוב?
- לא צריך להתחיל מחדש
- Claude זוכר את ההקשר
- חוסך זמן וכסף (פחות tokens)

## 📂 עבודה עם קבצים

### לראות קבצים
```
"תראה לי מה יש בתיקייה"
"מצא את כל קבצי ה-Excel"
"חפש קבצים שהשתנו היום"
```

### לקרוא קבצים
```
"תקרא את הקובץ README.md"
"מה כתוב בקובץ ההגדרות?"
"תראה לי את 10 השורות הראשונות של data.csv"
```

### לכתוב ולערוך
```
"צור קובץ חדש בשם notes.txt"
"הוסף שורה לסוף הקובץ"
"החלף את כל המילים 'ישן' ב'חדש'"
```

## ⚠️ דברים חשובים לזכור

### 1. Claude שואל לפני פעולות רגישות
```
Claude: "אני עומד למחוק 5 קבצים. להמשיך? (y/n)"
```
**תמיד תקראו לפני שמאשרים!**

### 2. אפשר לבטל
```
Ctrl+C  # עוצר פעולה באמצע
```

### 3. Claude לא יודע הכל
אם הוא לא בטוח, הוא יגיד. אל תניחו שהוא תמיד צודק.

## ✅ תרגיל מעשי

### משימה 1: ניווט (5 דקות)
1. פתחו Terminal
2. נווטו לתיקיית Documents
3. הפעילו `claude`
4. בקשו: "תראה לי מה יש פה"

### משימה 2: יצירת קובץ (5 דקות)
1. בקשו: "צור קובץ בשם test.txt עם הטקסט 'שלום עולם'"
2. בקשו: "תראה לי את תוכן הקובץ"
3. בקשו: "הוסף שורה 'זה עובד!'"

### משימה 3: חיפוש (5 דקות)
1. בקשו: "מצא את כל קבצי התמונות בתיקייה"
2. בקשו: "כמה קבצים יש בסך הכל?"

### משימה 4: יציאה וחזרה (5 דקות)
1. הקלידו `/exit`
2. הפעילו שוב `claude`
3. הקלידו `/resume`
4. וודאו שהשיחה הקודמת חזרה
