## 1. ניהול קבצים ונתונים (מתחיל עד בינוני)

### 1.1 ארגון ושינוי שמות קבצים
**רמת קושי:** מתחיל | **חיסכון זמן:** 30 דקות - 2 שעות

**דוגמאות אמיתיות:**
- ארגון 1,400+ צילומי מסך לתיקיות לוגיות לפי תוכן
- שינוי שמות חשבוניות מפורמט תאריך/שעה כאוטי לשמות סטנדרטיים
- מיון תיקיית downloads לפי סוג קובץ

**פרומפט:**
```
סקור את כל התמונות בתיקיית 'Screenshots', שנה שמות לפי תוכן,
קבץ לקטגוריות לוגיות, צור תת-תיקיות, והעבר תמונות בהתאם.
```

### 1.2 ארגון חשבוניות וקבלות
**רמת קושי:** מתחיל | **חיסכון זמן:** 1-3 שעות בחודש

**תהליך אוטומטי:**
- קריאת קבצי חשבוניות מתיקייה מבולגנת
- חילוץ: שם לקוח, תאריך, סכום
- שינוי שם: `Invoice_[Client]_[Date]_$[Amount].pdf`
- העברה לתיקיות חודשיות
- סימון למס

### 1.3 המרת מסמכים
**רמת קושי:** מתחיל | **חיסכון זמן:** 2-4 שעות לפרויקט

**דוגמאות:**
- המרת כל .docx ל-PDF
- דחיסת קבצי PDF
- המרת תמונות לפורמט PNG
- חילוץ טקסט מתמונות (OCR)

## 2. מחקר וניתוח תחרותי (בינוני עד מתקדם)

### 2.1 מחקר מתחרים אוטומטי
**רמת קושי:** בינוני | **חיסכון זמן:** 4-6 שעות לניתוח

**תהליך עבודה:**
```
לכל מתחרה ברשימה, נתח:
- אסטרטגיית תמחור
- סט פיצ'רים
- מסרים לקהל יעד
- מיצוב ייחודי

לאחר ניתוח, צור:
1. טבלת השוואת מחירים
2. מטריצת השוואת פיצ'רים
3. ניתוח פערי מיצוב
```

**תוצאה:** מה שלוקח 1-2 שבועות ידנית = 30-45 דקות עם Claude Code

### 2.2 מחקר שוק ומגמות
**רמת קושי:** בינוני | **חיסכון זמן:** 3-5 שעות לדוח

**שימושים:**
- זיהוי נקודות כאב של לקוחות
- מעקב אחר סנטימנט בתעשייה
- מציאת פערים בשוק
- מעקב אחר דיוני תמחור

## 3. אוטומציית אימייל ותקשורת (בינוני)

### 3.1 מיון וארגון תיבת דואר
**רמת קושי:** בינוני | **חיסכון זמן:** 1-2 שעות לסשן

**סיפור אמיתי:**
משתמש ניהל 10,000+ הודעות Gmail (4,000 לא נקראו):
- סיפק כללים למה לשמור/למחוק
- אחסון Gmail: 64% → 31% ב-1-2 שעות
- עכשיו מעבד אימייל ב-inbox zero

**פרומפט:**
```
עבור על תיבת הדואר שלי. לכל אימייל:
- מכירות/שיווק: ארכיון או מחיקה
- פניות לקוחות: סמן וסכם
- אדמיניסטרציה: ארגן לתיקיות לפי סוג
- שמור: אנשי קשר חשובים, חוזים, חשבוניות

צור סיכום משימות מהאימיילים שנשמרו.
```

### 3.2 יצירת תבניות אימייל
**רמת קושי:** מתחיל | **חיסכון זמן:** 30 דקות - שעה לתבנית

**דוגמאות:**
- רצפי קליטת לקוחות
- תבניות מעקב
- אימיילים להצעות שירות
- תגובות להתנגדויות

**תוצאה:** דוח ידני (3 שעות) → אוטומטי (15 דקות התחלתי, אח"כ חוזר)

### 4.2 אוטומציית דוחות SQL
**רמת קושי:** בינוני | **חיסכון זמן:** 1-2 שעות לדוח

**שימושים:**

- הכנסות לפי מוצר/סגמנט

- דוחות צינור מכירות

### 4.3 אוטומציית דוח הוצאות
**רמת קושי:** מתחיל | **חיסכון זמן:** 30 דקות - שעה לנסיעה

**תהליך:**
1. הורדת עסקאות כרטיס אשראי
2. הכנסה לתיקייה
3. הרצה: `claude "עשה דוח הוצאות. קטגר לפי סוג. צור דף אינטראקטיבי."`
4. קבלת: מעקב הוצאות אינטראקטיבי (10-20 דקות)

## 5. יצירת תוכן וכתיבה (מתחיל עד בינוני)

### 5.1 מחקר וסינתזה של תוכן
**רמת קושי:** מתחיל | **חיסכון זמן:** 1-2 שעות לפיסת תוכן

**דוגמה אמיתית:**
- משתמש אחד: 47 טיוטות → 12 פרסומים/חודש
- Claude מנתח רשומות יומן
- מזהה בעיות/דפוסים חוזרים
- מוצא דפוסים שהמשתמש היה "קרוב מדי לראות"

**תהליך:**
```

1. בעיות/תלונות חוזרות
2. דפוסים בחשיבה שלי
3. משימות שהזנחתי
4. הזדמנויות לצמיחה

צור: מתווה לפוסט בלוג בהתבסס על תובנות אלה
```

## 7. אוטומציה ואינטגרציית תהליכים (בינוני עד מתקדם)

### 7.1 משימות מתוזמנות עם Cron
**רמת קושי:** בינוני | **חיסכון זמן:** 1-2 שעות להקמת אוטומציה

**דוגמאות:**

**בדיקת איכות קוד יומית:**
```bash
0 9 * * * cd /path/to/project && claude "בדוק איכות קוד וצור דוח"
```

**סקירת בוקר בימי חול:**
```bash
0 8 * * 1-5 cd /path/to/project && claude "סקור שינויים מאתמול וצור הערות PR"
```

**עדכון מתחרים שבועי:**
```bash
0 9 * * 1 claude "הרץ פקודת /update-competitors"
```

### 7.2 אוטומציית GitHub
**רמת קושי:** בינוני | **חיסכון זמן:** 1-2 שעות לאוטומציה

**דוגמאות:**

**סקירת PR אוטומטית:**
```
/install-github-app
# Claude עכשיו סוקר אוטומטית כל PR, תופס שגיאות לוגיות שאנשים מפספסים
```

**תיוג Issues אוטומטי:**
```yaml
כש-issue חדש נוצר → Claude בודק → מקצה תוויות מתאימות
```

**תיקון Issues אוטומטי:**
כשמגיבים "@claude please fix this" → Claude:
1. קורא את ה-issue
2. מוצא קוד רלוונטי
3. כותב תיקון
4. פותח PR לאישור

### 7.3 אינטגרציות MCP (Model Context Protocol)
**רמת קושי:** מתקדם | **חיסכון זמן:** 2-3 שעות לאינטגרציה

**כלים זמינים (500+ דרך MCP):**
- אוטומציית Gmail
- תהליכי Slack
- ניהול issues ב-GitHub
- עדכוני בסיסי נתונים Notion
- פעולות גיליונות
- תזמון לוח שנה
- שאילתות בסיס נתונים
- קריאות API
